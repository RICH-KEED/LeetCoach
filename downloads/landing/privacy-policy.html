Privacy Policy

Last updated: June 17, 2026

Your privacy is of paramount importance to us. LeetCoach is designed to be a developer-centric tool that prioritizes transparency, local-first data integrity, and user control.

LeetCoach does not run central API servers, does not collect telemetry, and never stores or forwards your private code submissions or configured API keys to our own servers.

1. Information Collected & Used

LeetCoach functions entirely as a client-side web extension. We handle information in the following ways:

  • API Credentials: Your custom API keys (e.g., OpenAI, Gemini, Groq, OpenRouter) are stored locally on your machine using Chrome / Edge synchronized storage. They are never sent to any servers other than the API endpoint you explicitly configure in Settings.
  • LeetCode Submissions: Upon clicking the submit button on LeetCode, LeetCoach reads the problem context, description, programming language, code, runtime performance, and memory consumption. This context is used solely to form the prompt sent to your configured AI provider.
  • Local Cache: Review results are cached in your browser's local storage to save tokens and load subsequent views instantly. No external tracking databases are used.

2. Third-Party Services

When requesting an AI code review, the extension initiates direct network calls from your browser to the AI model provider endpoint you set up (such as OpenAI, Google, Groq, OpenRouter, or a local server running Ollama). Your data sent to these providers is subject to their respective terms and privacy policies:

  • OpenAI Privacy Policy
  • Google Gemini API Privacy Policy
  • Groq Privacy Policy
  • OpenRouter Privacy Policy

3. Data Retention & Deletion

Because all configuration data, cache files, and settings are stored locally inside the browser extension sandbox:

  • You can delete all stored settings and keys instantly by clicking "Reset" inside the extension settings page.
  • Uninstalling the LeetCoach extension from your browser immediately and permanently erases all associated local data and configurations.

4. Changes to This Policy

We may update this Privacy Policy from time to time. When we make updates, the version number of the extension is incremented, and updates are detailed in the "What's New" changelog page upon installation.

5. Contact Us

If you have any questions or feedback regarding this policy or the security of your data in LeetCoach, feel free to contact us via our GitHub repository.
