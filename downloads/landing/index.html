class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        # Optimized two-pass with a hashmap
        prevMap = {} # val -> index
        
        for i, n in enumerate(nums):
            diff = target - n
            if diff in prevMap:
                return [prevMap[diff], i]
            prevMap[n] = i
            
        return [] # No solution found

Algorithm Summary

You solved this using a Single-Pass Hash Map (complement mapping). By caching previous values and checking for target - n, you perform lookups in O(1) instead of a nested O(N²) brute force scan.

Complexity

Time Complexity: O(N)
Space Complexity: O(N)

Your runtime is 36ms, beating 88.4% of Python submissions. The hash map ensures we only traverse the array once.
